## ⚡ Overview
**HealthRisk AI** is a highly accurate Machine Learning dashboard designed to predict cardiovascular risk based on patient vitals and demographics. Built with a full MLOps pipeline, zero-leakage preprocessing, and an advanced **XGBoost predictive engine**, the system achieves a ROC-AUC score of **90.7%**.

## 🔬 Enterprise Features
*   🧬 **Predictive Engine:** Advanced XGBoost model trained on deep clinical correlations (Blood Pressure, Heart Rate, BMI).
*   🛡️ **Zero-Leakage Pipeline:** Industrial preprocessing pipelines built with `scikit-learn` ColumnTransformers.
*   💻 **Interactive UI:** Multi-page dashboard with a secure real-time patient registry, native CSV exporting, and simulated EHR network syncing.
*   شف **Algorithmic Transparency:** Explainable AI methodology providing clear clinical risk factors for every decision.

## 🚀 Installation & Running Locally

1. **Clone the repository:**
```bash
git clone https://github.com/atharvshukla76/HealthRisk-AI.git
cd HealthRisk-AI
```

2. **Install dependencies:**
```bash
pip install -r requirements.txt
```

3. **Run the Dashboard:**
```bash
streamlit run frontend/streamlit_app.py
```
